# HBuilder 混合开发项目
##1.介绍
1.软件只有Android版本
2.源码已经开源，接口请自行测试

## 2.目录结构说明
```
    css                 //样式文件夹
    fonts               //字体文件夹
    images              //图片等资源文件夹
    js                  //javascript文件夹
    pages               //页面文件夹
    guidePage.html      //引导页
    index.html          //app入口页面
    main.html           //首页（主要的）
    manifest.json       //app的配置文件
```
## 3.已上线功能
1.翻译（单词句子翻译）
2.身份证归属地查询
3.手机号码归属地查询
4.银行卡归属地查询
5.获取手机设备信息
6.智能问答
7.标准身材计算
8.IP地址查询
